🚨 필수 공지사항
  • 게임의 재미를 30배 올려줄 Magic 아이템 추가 모드인 Epic Loot 플러그인은 별도의 한글화 작업이 반드시 필요합니다.
  • 일단 아래 파일을 다운로드받습니다.
💾  발헤임 Epic Loot 한글화 파일
  • 다운로드 받은 파일을 본인의 컴퓨터에 설치해야합니다.
  • r2modman을 켜고, 왼쪽의 setting을 누르고, Browse profile folder를 누릅니다.
  • 그 후, 열리는 파일 탐색기에서 BepInEx/plugins/RandyKnapp-EpicLoot로 이동합니다.
  • 다운로드 받은 파일을 해당 위치에 붙혀넣기 합니다. (magiceffects.json) 덮어쓰기하냐고 물어보면 덮어쓰기합니다.
  • 완료후 r2modman의 Start modded를 눌러합니다
